## Setup

1. Clone and install dependencies: <br/>
```sh
# Clone the repo
$ git clone https://github.com/AZYHAR/Wishlist.git

# Navigate to cloned folder and install dependencies
$ cd wishlist
$ npm install
```
<br/>

2. Start expo development server<br/>


```sh
$ expo start
```
